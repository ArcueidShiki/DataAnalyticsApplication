# Installation

```bash
## python version 3.12.9
python -m venv venv

# For Linux/MacOs
source venv/bin/activate

# For windows
venv\Scripts\activate.bat

pip install pre-commit flask black isort bandit yfinance flask-restful flasgger pytest selenium locust flask_sqlalchemy flask_jwt_extended flask_cache flask_limiter

## to run
flask run

## or

python app.py
```
